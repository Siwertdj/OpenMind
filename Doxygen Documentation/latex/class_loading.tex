\doxysubsection{Detailed Description}
This class is responsible for loading all the \doxyref{Save\+Data}{p.}{class_save_data} from the save file. 

There are checks for all relevant exceptions. After an exception is thrown, it results in an error message in the debug menu. A check is made for the location of the notebook file. If it does not exist, an error message will appear in the debug menu and the loading process fails.
