void Saving.\+Save (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Saves data to a file. All data saved to a file is stored in \doxyref{Save\+Data}{p.}{class_save_data}$>$. When accessing the notebook file, all relevant exceptions are caught and result in an error in the debug console and results in nothing being saved. When the specified directory and or file to save to cannot be found, these will be created anew. 

This method results in an error in the debug console, if the gamemanager is not loaded, after which it will exit the function.
